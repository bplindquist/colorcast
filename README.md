# Colorcast

A minimalist weather app with color-coded data visualization using a rainbow theme.

## Features

- Search weather by zip code
- Color-coded weather metrics (temperature, humidity, UV index, wind, dew point)
- Responsive design for mobile and desktop

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- [@heeler/ui](https://www.npmjs.com/package/@heeler/ui) components
